# Telangana Weather Data Analysis Dashboard

## **Project Overview**
This project involves analyzing weather patterns across Telangana using Power BI. The dashboard provides clear insights into temperature, humidity, and rainfall trends, helping stakeholders make informed decisions for planning, agriculture, and policy.

## **Tools & Technologies**
- **Power BI:** Interactive dashboards, data visualization, DAX formulas

## **Key Features**
- **Interactive Dashboard:** Allows filtering by region, month, and year  
- **Trend Analysis:** Monthly and seasonal analysis of temperature, humidity, and rainfall  
- **KPI Cards:** Highlight key metrics for quick insights  
- **Visual Insights:** Easy-to-understand charts and maps for data-driven decisions

## **Outcome / Impact**
- Enabled stakeholders to **identify weather trends and regional variations** efficiently  
- Supported **decision-making** in agriculture and policy planning  
- Dashboard can be **updated with new data** for continuous monitoring
